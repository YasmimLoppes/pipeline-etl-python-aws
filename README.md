Pipeline de ETL: Automação de Dados com Python & AWS S3
Este projeto demonstra a automação de um pipeline de Engenharia de Dados, realizando o tratamento de arquivos CSV e a persistência em nuvem.
🛠️ Tecnologias
* Linguagem: Python (Pandas)
* Cloud: AWS S3 (Simple Storage Service)
* Ambiente: VS Code
🚀 O que este projeto faz:
* Extração: Leitura de dados brutos de arquivos CSV.
* Transformação: Limpeza e manipulação de dados utilizando a biblioteca Pandas.
* Carga: Upload automatizado dos dados tratados para um Bucket na AWS S3.
